# 民宿预约管理系统 - 项目记忆

## 项目概况
- 项目名称：民宿预约管理系统
- 技术栈：前端 Vue3 + TypeScript，后端 Spring Boot，数据库 MySQL
- 项目位置：/Users/mini/Downloads/民宿预约系统/
- 原型文件：preview (3).html（可交互单页原型，含用户端+管理端完整流程）

## 核心业务
- 2个角色：普通用户（旅客）、后台管理员
- 用户端9个页面：首页、房源列表、房源详情、预约下单、预约成功、我的订单、订单详情、发表评价、个人中心、我的收藏
- 管理端8个模块：后台登录、数据看板、房源管理、房态维护、订单管理、评价管理、用户管理、系统配置
- 订单9种状态：待确认→待支付→已预订→入住中→已完成→已评价；异常分支：已取消、退款中→已退款
- 10个数据对象：User/House/Order/Review/Favorite/Availability/Admin/City/Tag/SystemConfig

## 已确认关键决策（v1.1）
- 用户端登录：用户名+密码，不需要手机号
- 禁用用户：禁止登录
- 管理员：单一账号，无权限分级
- 后台登录：无验证码
- 评价图片上传：需要实现
- 房源封面：支持真实图片上传（cover_image 字段）
- 分页：每页固定20条
- 优惠：由系统配置驱动，无优惠券系统
- 待支付超时：30分钟自动取消（定时任务）
- 入住中状态：入住日期当天系统自动流转（定时任务）
- 评分计算：只统计"显示"状态评价
- 联系房东：保持占位，不实现
- 数据看板：需要真实接口
- 收藏：保存到后端数据库
- 通知：不发送短信/邮件
